MCK-RA8T2 Renesas Flexible Motor Control Kit for RA8T2 MCU Group

RTK0EMA6L0S00020BJ

The MCK-RA8T2 is a development kit designed for easy evaluation of motor control using permanent magnet synchronous motors (brushless DC motors). This kit includes the RA8T2 CPU board ( MCB-RA8T2 ), an inverter board ( MCI-LV-1 ), a permanent magnet synchronous motor, and accessories such as cables and standoffs. Start motor control evaluation with the RA8T2 by using this kit together with the sample code and QE for Motor , both available for download from the website.

Product Specification

    MCK-RA8T2
    • The CPU board is equipped with the RA8T2 as the motor control MCU.
    • Inverter board for 3-phase BLDC motor (48V/10A rating)
    • On-board debugger for flash programming of RA8T2
    • Supports 1-shunt/3-shunt current sensing
    • Overcurrent detection
    • Supports the motor control development support tool, Renesas Motor Workbench ( Introduction to the use of Motor Workbench (with videos) )
    • Motor control evaluation and debugging can be performed safely with the communication board which isolates PC and the CPU board electrically
    • Supports Hall sensors, encoders, and inductive position sensors
    • Included items
      • RA8T2 CPU board (MCB-RA8T2)
      • Inverter board (MCI-LV-1)
      • Communication board (MC-COM)
      • Permanent magnet synchronous motors
      • Accessories (cables, standoffs, etc.)
