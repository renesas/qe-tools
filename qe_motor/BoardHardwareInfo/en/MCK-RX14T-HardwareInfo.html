MCK-RX14T Renesas Flexible Motor Control Kit for RX14T MCU Group

RTK0EMXH30S00020BJ

The MCK-RX14T is a development kit that enables easy evaluation of motor control using permanent magnet synchronous motors (brushless DC motors). With this product and the sample code and QE for Motor that can be downloaded from the website, you can start evaluating motor control using the RX14T right away.

Note in using sample program

    The setup of hardware required to execute the sample program depends on the device version and the motor control type. Please refer to "Hardware Setup" and the application note of each sample program for the setup of hardware required for sample program.
