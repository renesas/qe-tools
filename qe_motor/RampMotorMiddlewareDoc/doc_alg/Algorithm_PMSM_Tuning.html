PM Motor Tuning

1. PM Motor Parameters

To control a PM motor, the following parameters (motor constants) are required:

PM Motor Parameter Symbol Unit Description
Resistance Ra (R) Ω Winding resistance
d-axis inductance Ld H Ease of current flow between winding and magnet (d-axis)
q-axis inductance Lq H Ease of current flow between winding and magnet (q-axis)
Flux linkage / Back-EMF constant Ke, Ψ(Φf) Wb Magnitude of induced voltage due to magnet rotation
Moment of inertia J kg·m² Mechanical inertia
Viscous friction coefficient D Nm·s/rad Mechanical friction

4. Measurement Using Electrical Instruments

4.1 Measuring R, Ld, Lq with an LCR Meter

  • Use a calibrated LCR meter and perform open/short adjustments
  • Measure Rs (series resistance) and Ls (series inductance)
  • Measurement frequency: around 100 Hz (near the motor’s fundamental operating frequency)

Procedure:

  1. Place probes across U–V (also V–W, W–U)
  2. Read Rs and divide by 1/2 → yields motor resistance Ra
  3. Slowly rotate rotor and record minimum inductance (Ls-min) and maximum inductance (Ls-max):
    • Ls-min = 2Ld → Ld = Ls-min / 2
    • Ls-max = 2Lq → Lq = Ls-max / 2
  4. Confirm across other phases as well

4.2 Measuring Flux Linkage Ψ (Φf) Using an Oscilloscope

  • Rotate motor at a constant speed using a power tool, etc., and observe induced voltage
  • Measure frequency f and peak voltage Vp
  • Note: Line-to-line voltage is √3 times larger and phase-shifted by 30°

\[\Phi_f = \frac{V_p}{\sqrt{2} \cdot \omega} = \frac{V_p}{\sqrt{2} \cdot 2\pi f}\]

Set the measured Ψ into the PMFOC motor parameter “magnetic flux” in FSP properties.

4.3 Calculating Inertia J

  • Often listed in the motor datasheet/UM (otherwise contact manufacturer)
  • If a load is connected, sum the following:
    • Motor inertia
    • Coupling
    • Shaft (non-motor components)
    • Load-side inertia

5. Parameter Acquisition Algorithms and Flow

5.1 Overview

Parameter Algorithm Measurement Method
R Two-point difference / RLS / Fourier integral DC voltage injection → estimate from current
Ld, Lq RLS / Fourier integral 200 Hz AC voltage injection → estimate from current
Ψ (Emf, Ke) Back-EMF observer Accelerate up to 25% supply voltage → remeasure at 50% speed
J, D RLS / Fourier integral Estimate from torque during ±15% speed variation around 50% max speed

5.2 Measurement Flow

  1. Inject DC voltage → measure current → estimate R
  2. Inject AC voltage → measure current → estimate Ld
  3. Inject AC voltage → measure current → estimate Lq
  4. Motor drive → Back-EMF observer → estimate Ψ (Ke)
  5. Motor drive with speed variation → measure current → estimate J, D

τ (torque) → [1/(JS+D)] → ωm → [1/s] → θm

Mechanical equation:

\[J\frac{d\omega_m}{dt} + D\omega_m = T_e - T_L\]

7. Measurement of Ra, Ld, Lq Using the Fourier Integral Method (Static Method)

7.1 Principle

Fix the rotor d‑axis to the U‑phase axis (ω = 0), then apply an AC voltage to the d‑axis:

\[V_{dref} = E\sin\omega t \quad \cdots (1)\]

Under these conditions, the equivalent circuit becomes a simple RL series (no induced EMF), and the current is:

\[i(t) = \frac{E}{\sqrt{R_a^2 + (\omega L_d)^2}} \left\{ \sin(\omega t - \varphi) + e^{-\frac{R_a}{L_d}t} \sin\varphi \right\} \quad \cdots (2)\]

where
\[\varphi = \tan^{-1}\dfrac{\omega L_d}{R_a}\]

7.2 Fourier Integral Expansion

Expansion of \[i \cdot \sin\omega t\]:

\[i \cdot \sin\omega t = \frac{E}{2\sqrt{R_a^2+(\omega L_d)^2}}\{-\cos(2\omega t-\varphi)+\cos\varphi\}\]
\[+ \frac{E}{2\sqrt{R_a^2+(\omega L_d)^2}} e^{-\frac{R_a}{L_d}t}\{-\cos(\omega t+\varphi)+\cos(\omega t-\varphi)\} \quad \cdots (3)\]

Expansion of \[i \cdot \cos\omega t\]:

\[i \cdot \cos\omega t = \frac{E}{2\sqrt{R_a^2+(\omega L_d)^2}}\{\sin(2\omega t-\varphi)-\sin\varphi\}\]
\[+ \frac{E}{2\sqrt{R_a^2+(\omega L_d)^2}} e^{-\frac{R_a}{L_d}t}\{\sin(\omega t+\varphi)-\sin(\omega t-\varphi)\}\]

7.3 Extraction of DC Components by Integral Averaging

After sufficient time (transient term = 0), integrate over several cycles of frequency ω; the AC terms cancel, leaving only DC components:

\[A_{sin} = \frac{E R_a / 2}{R_a^2 + (\omega L_d)^2} \quad \cdots (4)\]

\[A_{cos} = \frac{-E \omega L_d / 2}{R_a^2 + (\omega L_d)^2} \quad \cdots (6)\]

7.4 Calculating Ra and Ld

\[R_a = \frac{E A_{sin}}{2(A_{sin}^2 + A_{cos}^2)}\]

\[L_d = \frac{E A_{cos}}{2(A_{sin}^2 + A_{cos}^2)}\]

Lq is obtained in the same way by applying AC voltage to the q‑axis.

8. Measurement of Ra, Ld, Lq Using Recursive Least Squares (RLS)

8.1 Difference Equation Derivation

For the d‑axis RL series:

\[i_d = \frac{1}{L_d S + R_a} v_d \quad \cdots (1)\]

Discretizing the derivative:

\[i_d(k) = \frac{L_d/T_s}{L_d/T_s + R_a} i_d(k-1) + \frac{1}{L_d/T_s + R_a} v_d(k-1) \quad \cdots (2)\]

Thus:

\[i_d(k) = A\, i_d(k-1) + B\, v_d(k-1) \quad \cdots (3)\]

Matrix form:

\[i_d(k) = [A, B] \begin{bmatrix} i_d(k-1) \\ v_d(k-1) \end{bmatrix} \quad \cdots (4)\]

8.2 Least Squares Model

\[Y(k) = \theta^T \varphi(k)\]

  • \[Y(k) = i_d(k)\]
  • \[\theta^T = [A, B]\]
  • \[\varphi(k) = [i_d(k-1), v_d(k-1)]^T\]

Prediction error:

\[\varepsilon(k,\theta) = Y(k) - \theta^T\varphi(k)\]

8.3 Cost Function and Minimization

\[J_N(\theta) = \frac{1}{N}\sum_{k=1}^{N} \{Y(k)-\theta^T\varphi(k)\}^2\]

Expanding:

\[J_N(\theta) = \theta^T R(N)\theta - 2\theta^T f(N) + C(N)\]

Where:

\[R(N) = \frac{1}{N}\sum_{k=1}^{N}\varphi(k)\varphi^T(k)\] \[f(N) = \frac{1}{N}\sum_{k=1}^{N}\varphi(k)Y(k)\] \[C(N) = \frac{1}{N}\sum_{k=1}^{N}Y^2(k)\]

Setting derivative = 0:

\[\hat{\theta}(N) = R(N)^{-1} f(N)\]

8.4 Recursive Form (RLS)

Applying matrix inversion lemma:

\[P(N) = P(N-1) - \frac{P(N-1)\varphi(N)\varphi^T(N)P(N-1)}{1 + \varphi^T(N)P(N-1)\varphi(N)}\]

Parameter update:

\[\hat{\theta}(N) = \hat{\theta}(N-1) + \frac{P(N-1)\varphi(N)}{1+\varphi^T(N)P(N-1)\varphi(N)}\varepsilon(N)\]

Initial values:

\[\hat{\theta}(0)=\begin{bmatrix}0\\0\end{bmatrix},\quad P(0)=\begin{bmatrix}\gamma & 0 \\ 0 & \gamma \end{bmatrix},\ \gamma > 1000\sim10000\]

8.5 RLS with Forgetting Factor (for Time-Varying Systems)

Cost function:

\[I_N(\theta)=\sum_{k=1}^{N}\lambda^{N-k}\varepsilon^2(k),\quad \lambda=0.97\sim0.995\]

Updates:

\[\hat{\theta}(N)=\hat{\theta}(N-1)+\frac{P(N-1)\varphi(N)}{\lambda+\varphi^T(N)P(N-1)\varphi(N)}\varepsilon(N)\]

\[P(N)=\frac{1}{\lambda}\left\{P(N-1)-\frac{P(N-1)\varphi(N)\varphi^T(N)P(N-1)}{\lambda+\varphi^T(N)P(N-1)\varphi(N)}\right\}\]

8.6 Recovering Ra and Ld

From RLS‑estimated A and B:

\[L_d = \frac{A}{B}T_s,\quad R_a = \frac{1-A}{B}\]

q‑axis:

\[L_q = \frac{C}{D}T_s,\quad R_a = \frac{1-C}{D}\]

9. Measurement of Flux Linkage Ψ (Φf) Using the Back‑EMF Observer

9.1 Back‑EMF Observer Block Diagram

d‑axis observer:

  • Input: \[v_d^*\]
  • Model: \[\frac{1}{L_d s}\] → estimated current \[\hat{i}_d\]
  • Feedback: R, \[L_d K_{Ed1}\]
  • Integrator → disturbance estimate:
    \[\hat{d}_d = \omega^* L_q i_q - e_d\]

q‑axis observer:

  • Input: \[v_q^*\]
  • Model: \[\frac{1}{L_q s}\]
  • Integrator → disturbance estimate:
    \[\hat{d}_q = -\omega^* L_d i_d + e_q\]

9.2 Computing Induced Voltage

\[E_d = -\hat{d}_d + \omega L_q i_q \cong \omega\Phi_f\sin\Delta\theta\]

\[E_q = -\hat{d}_q - \omega L_d i_d \cong \omega\Phi_f\cos\Delta\theta\]

9.3 Computing Flux Linkage (Back‑EMF Constant)

\[\Phi_f = \frac{E}{\omega} = \frac{\sqrt{E_d^2 + E_q^2}}{\omega}\]

10. Measurement of J and D (RLS / Rotational Method)

10.1 Mechanical Difference Equation

For no‑load (TL = 0):

\[\omega_m(k) = [E, F] \begin{bmatrix}\omega_m(k-1)\\P_n\Phi_f i_q(k-1)\end{bmatrix}\]

Expanded:

\[\omega_m(k)=\frac{J/T_s}{J/T_s + D}\omega_m(k-1)+\frac{1}{J/T_s + D}P_n\Phi_f i_q(k-1)\]

10.2 Recovering J and D

After estimating E and F via RLS:

\[J = \frac{E}{F}T_s P_n\Phi_f,\quad D = \frac{1-E}{F}P_n\Phi_f\]
